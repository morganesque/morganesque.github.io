This was a great project. It was an absolute pleasure to work with Monica & Matt from Kilo75. We started from personas, working these up into user flows and wireframes. Then within the tight restrictions of the existing brand we designed a crisp clean but lively and bustling website which presents visitors with the most interesting and useful content up front and gives them easy access to everything they’d need to know about the markets.
